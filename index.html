Q1. How are the flash cells encoded in binary bits for Multi-Level Cell (MLC) schemes?

Ans: In 2-bit MLC, it is generally defined as four distinct threshold voltage windows (distributions). Each distribution corresponds to a 2-bit code.

In most cases the states are mapped in the following way:

Vth States

2-bit Code

Description

L0

11

Lowest Vth Range

L1

01

Second-Lowest Vth Range

L2

00

Second-highest Vth Range

L3

10

Highest Vth Range

 

When storing data in an MLC NAND flash cell, you must perform two separate write operations for each cell: one for the Least Significant Bit (LSB) page, and one for the Most Significant Bit (MSB) page. As a simple example, let's look at the first four cells of a page in the memory. Suppose the LSB page data for these cells is [1,1,0,0] and the MSB page data is [1,0,0,1]. First, the LSB bits are programmed, placing cells into two broad voltage levels. Next, the MSB bits are written, refining each cell to one of four possible threshold-voltage (Vth) states. Q2. After the Shared LSB Page is written what will be the state of flash cells initially then what about right after the MSB Page write operation?

Ans: When programming MLC, the flash control logic often writes bits in two steps (or two separate program operations) to achieve the final threshold distribution. First, it programs the LSBs (coarse placement), and then it programs the MSBs (fine placement) later.

After LSB Page write by the bitstream 1100:

Divides the four final states into two broad groups. It is a quick program operation that sets the cell to either erased or programmed range.

After MSB Page write by the bitstream 1001:

In the second program pass, the controller refines the threshold voltage of each cell to its final one of the four states. The final states (L0, L1, L2, L3) now encode both the LSB and the MSB bits (two bits total).

Q3. How many senses are the amplifier reading operations required to read the LSB page as well as MSB page?

Ans:

LSB Page Read: The data "1 1 0 0" (LSB page) is written, and a single sense amplifier read operation can verify if the cells are in one of two voltage ranges (associated with "1" or "0" for the LSB).

 

MSB Page Read: The MSB page data "1 0 1 0" further adjusts the threshold voltages. To decode both the MSB and LSB data accurately, additional 2 more sense amplifier read operations are required to distinguish between the four possible states ("11," "10," "01," and "00").
